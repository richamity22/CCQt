[1] Uncompress this ZIP file into a folder.

[2] Double-click on "CC_Qt.exe" to start the program

[3] Open the test file: in the menu, click on "File -> Open", browse to "test.ged"

[4] Change the skin: click on "File -> Skins", browse to one of the skins (file ending in ".QSS"), select the skin.

[5] Copy one of the CSS skin files to make your own working copy. Change your CSS file to the color scheme you have designed, send me a screen shot (image).
